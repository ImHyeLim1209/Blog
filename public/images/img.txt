Next.js는 이미지 최적화를 자동으로 지원한다.
 - 이미지가 뷰포트 밖에 있는 경우 기본적으로 지연 로드된다.
 - WebP와 같은 최신 형식으로 이미지 크기를 조정하고, 최적화 하고 제공한다.-> 작은 뷰포트에 큰 이미지가 로딩되는 것 방지
 - 로컬 이미지 사용 시 width, height 크기의 자동 계산
 - 원격 이미지의 경우 Next.js가 빌드 중에 원격 파일에 접근할 수 없으므로 width, height를 제공해주어야 한다.
   - 원격 이미지에 이미지 최적화를 사용하고 싶다면 next.config.js 파일에 설정한다.
 - 이미지 로드 시, 우선순위를 지정해줄 수 있다.
 - 레이아웃 시프트 : 이미지가 로드될 때 페이지의 다른 요소를 밀어내는 현상
  - 로컬파일 로드 시 width, height를 지정하지 않고 자동으로 하거나, 잘 계산해서 명시적으로 지정한다.
  - layout=fill을 이용하여 이미지가 부모 요소를 채우도록 확장한다.
